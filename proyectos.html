OliCam

Aplicación Android que muestra el vídeo de la cámara a pantalla completa sin botones o controles que obstaculicen la visión. No captura imágenes ni vídeos. Destinada a streamers que quieran capturar la pantálla del dispositivo.

HBO Hotkeys

Extensión de chrome que añade controles por teclado al reproductor integrado de la plataforma HBO. Utiliza DOM y JavaScript nativo para mayor compatibilidad.

GarageDoor

Aplicación Android que permite abrir una puerta de garaje conectada a una raspberry. Servidor Node con autenticación a través de Firebase. Guarda registros en una base de datos integrada con automatización para la baja de usuarios.

Tile Map

Aplicación Android que permite guardar la posición en la que has aparcado y otener indicaciones de google maps para volver a ella. Destaca por soportar acciones en un clic a través de "Quick Settings Tiles". En desarrollo.

Spotify Sync

Extensión de Chrome que permite sincronizar automáticamente la reproducción de música de Spotify. El usuario administrador puede iniciar una sala a la que otros usuarios acceden a través de un enlace. En desarrollo.
